## Building & Publishing Library and package.

:::note 
As a contributor you don't need to do this But maintainers are required to use these commands.
:::

1. Build the Library useing `cargo build` else if npm package run `npjs-cli build`. 

2. Publish the Library using `cargo publish` else if npm package run `npm publish`.
