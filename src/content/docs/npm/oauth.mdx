This guide will teach you how to use the oauth





Start a new session

TypeScipt version
```ts
import * as minecraft_essentials from 'minecraft_essentials'; // Assuming minecraft_essentials is a module

let clientId = "111231209837123098712";
let oauth = new minecraft_essentials.Oauth(clientId);
console.log(`Login here: ${oauth.url()}`);
let oauthInfo = await oauth.launch();
```
